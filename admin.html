📋 Инструкция (один раз):

1. Зайдите на supabase.com → создайте бесплатный проект
2. В левом меню → SQL Editor → вставьте и выполните:
create table wedding_guests ( id bigint generated always as identity primary key, name text, attendance text, food text, drinks text, submitted_at timestamptz default now() ); -- разрешить запись всем (для формы на сайте) create policy "insert" on wedding_guests for insert with check (true); -- разрешить чтение/удаление только с ключом create policy "select" on wedding_guests for select using (true); create policy "delete" on wedding_guests for delete using (true); alter table wedding_guests enable row level security;
3. Project Settings → API → скопируйте Project URL и anon / public key
4. Вставьте их в поля ниже и в index.html замените плейсхолдеры
💡 URL и ключ сохранятся в браузере — вводить повторно не нужно.
На другом устройстве — ввести один раз снова.
